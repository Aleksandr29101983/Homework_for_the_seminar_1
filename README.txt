Дано домашнее задание: Написать программу вычисления n-ого треугольного числа.

Общая формула для n-го по порядку треугольного числа: Tn = (1/2)*n*(n+1), n = 1, 2, 3...;
Последовательность треугольных чисел Tn бесконечна.

Произведена декомпозиция решения задачи на методы:
1. Получение данных от пользователя.
2. Вычисление n-го треугольного числа Tn по приведенной формуле.
3. Вывод полученного n-го треугольного числа Tn в консоль.

После создания данных методов создан метод управления main, в котором прописан алгоритм использования остальных методов
